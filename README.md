# testQiwi
Setting Adaptive layout for the company Qiwi

Сверстать адаптивную веб-страницу.

При уменьшении размеров окна фоновые изображения не должны исчезать.

Элементы страницы должны иметь отступ от левого и правого края 20px на всех
разрешениях (не отностится к фоновым изображениям).


Браузеры: Chrome, Firefox, Opera, IE8+.

Минимальная ширина: 360px.

Максимальная ширина: 1920px.

Шрифт: PT Sans
