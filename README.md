# testQiwi
Setting Adaptive layout for the company Qiwi

Верстка адаптивной веб-страницы.
При уменьшении размеров окна фоновые изображения не должны исчезать.
Элементы страницы должны иметь отступ от левого и правого края 20px на всех
разрешениях (не отностится к фоновым изображениям).
Браузеры: Chrome, Firefox, Opera, IE9+.
Минимальная ширина: 320px.
Максимальная ширина: 1920px.
Шрифт: PT Sans

Использовал препроцессор Stylus, адаптивные макет делал с помощью Jeet, так же использовал Nib для Stylus. 
Отказался от поддержки IE8 так как очень хотелось попробовать связку Jeet + Stylus
К сожалению не было возможности достаточно поротестировать непосредственно на устройствах.
